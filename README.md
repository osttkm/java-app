このレポジトリは学部生2年次後期に取り扱ったjavaアプリのアーカイブです．
シューティングゲーム等講義で取り扱ったコードも載せてあります．最終発表課題ではお絵かきアプリを作成．コードはfinal_workにあります．
講義で取り扱ったシューティングゲームの開発の際に自分の欲しい画像のgifがないという声が学生の中であったため，それを解決するために自作の
アイコンや拾ってきた画像を使いやすいように編集するアプリを作成しようと決めました．最終課題に関しては提出した動画も載せてあります．

layerの機能やgif保存機能を付けました．gif画像保存について，デフォルトでは白背景を透過します．
透過してほしくない部分に関してはprotect機能により矩形で指定することが可能です．
